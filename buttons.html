HTML Button Types

In HTML, the <button> element can have different types based on the type attribute. Each type serves a specific purpose.

1. type="button"

Purpose: A general-purpose button that does not have any default behavior. Typically used with JavaScript.

2. type="submit"

Purpose: Submits the form data to the server.

3. type="reset"

Purpose: Resets all form fields to their initial values.

4. No type Attribute

Default behavior:

  • Inside a form → acts as submit.
  • Outside a form → acts as button.

Additional Notes: Buttons can be styled with Bootstrap’s button classes (.btn-primary, .btn-success, etc.). Always ensure accessibility with meaningful text or aria-label.
