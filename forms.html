<form> Tags are essential for collecting user input on web pages. They incorporate a variety of interactive controls such as text fields, numeric inputs, email fields, password fields, checkboxes, radio buttons, and submit buttons. Over 85% of websites rely on forms to gather data from users, making them a fundamental component of modern web development.

Syntax:

<form>
   <!--form elements-->
</form>

HTML Form

Form Elements

Elements Descriptions
<Label> Defines labels for <form> elements.
<Input> Used to get input data (text, password, email, etc.)
<Button> Defines a clickable button.
<Select> Creates a drop-down list.
<Textarea> Accepts long text input.
<Fieldset> Groups related form elements with a box.
<Legend> Caption for a fieldset.
<Datalist> Specifies pre-defined list options.
<Output> Displays calculation results.
<Option> Defines options in a drop-down.
<Optgroup> Groups related drop-down options.

Commonly Used Input Types

Input Type Description
<input type="text"> One-line text input field
<input type="password"> Password input field
<input type="submit"> Submit button
<input type="reset"> Reset button
<input type="radio"> Radio button
<input type="email"> Validates email input
<input type="number"> Numeric input with range options
<input type="checkbox"> Multiple choice selection
<input type="date"> Date picker
<input type="time"> Time picker
<input type="file"> File upload input
