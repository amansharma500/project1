Basic Structure of HTML

Topics we are going to learn:

  1. html -- tag
  2. head -- tag
  3. body -- tag
  4. title -- tag
  5. style -- tag
  6. script -- tag
  7. link -- tag
  8. meta -- tag
  9. class -- attribute
  10. id -- attribute

Basic Structure:

  1. <html>...</html> – Parent tag for <head> and <body>.
  2. <head>...</head> – Contains metadata, styles, and scripts.
  3. <body>...</body> – Contains the main content of the webpage.
  4. "! + Enter Key" – Shortcut for auto-generating HTML boilerplate.
  5. Emmet plugin improves HTML & CSS workflow.
  6. Alt + Shift + ↓ – Duplicate a line in VS Code.
  7. Ctrl + Space – Suggestions for attributes/values.

Example:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Html Document Structure</title>
</head>
<body>
    // Main Content
</body>
</html>

Class & ID

  1. Both are used to target/select HTML elements.
  2. .classname for class selector.
  3. Multiple classes allowed: <p class="Red Text">.
  4. #idname for ID selector.
  5. Classes can repeat, IDs must be unique.
